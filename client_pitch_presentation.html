16. TECHNOLOGY DECISION

WordPress vs. Proposed Serverless Stack

A direct, transparent comparison showing why a monolithic WordPress setup would expose EGYGRAFX to unnecessary operational overhead and security threats:

Operational Metric Legacy WordPress Setup Proposed Serverless Next.js Stack
Speed & Load Time Slow database queries per load (High B2B bounce risk). Blazing Fast pre-rendered static edge page files.
Security & Hacking High threat from buggy third-party plugins. Ironclad. No public database is exposed to routers.
Database Flexibility Rigid schemas struggle with complex B2B links. Flexible. Firestore NoSQL maps products relationally.
Support Maintenance High. Requires monthly SLAs for patches and updates. Zero. Runs indefinitely with zero active support.
Ongoing Hosting Costs `$25 - $80 / month` for dedicated servers. `$0 to $20 / month` total running costs.
