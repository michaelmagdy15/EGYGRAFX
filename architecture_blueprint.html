2 Programmatic SEO & Routing Blueprint

By mapping out static, dynamic-slug folders, Next.js performs background Incremental Static Regeneration (ISR) to compile dynamic landing pages. Every supplier brand, application sector, and machinery model is automatically assigned its own semantic URL path to target high-intent B2B search terms on Google.

    AppRoute --> ISR{SSG Cache Valid?}
    BrandRoute --> ISR
    ProdRoute --> ISR

    ISR -->|YES| Static[Serve High-Performance static HTML/CSS]
    ISR -->|NO| Regeneration[Background Incremental Static Regeneration - 60s]
    Regeneration --> Firestore[Query Firestore CMS collections]
    Firestore --> Render[Re-render page layout on server]

    StaticCache --> Static

4 Visual Rebranding Synthesis

Applying the Xerox branding transformation to EGYGRAFX, the company moves from a rigid, uppercase, sharp serif profile (which represents high B2B friction) to a highly approachable, lowercase, rounded geometric sans-serif identity. The traditional four-color offset CMYK blocks are replaced with a sleek overlapping infinity loop sphere that captures the movement of high-speed roll systems and unified printing intelligence.

Legacy Identity (Current)
Key Visual Friction Points:
  • Sharp, high-contrast serif typography in all-caps.
  • Literal CMYK offset-printing color grid (narrow market scope).
  • Traditional, rigid corporate appearance.
Proposed B2B Rebrand
Key Strategic Growth Benefits:
  • Modern lowercase rounded geometric sans-serif for warmth.
  • Overlapping infinity curves representing roll flow & intelligence.
  • Warm Crimson Red & Digital Coral tones indicating energetic tech.
